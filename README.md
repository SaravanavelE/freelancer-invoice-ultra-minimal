# Freelancer Invoice Generator (Ultra Minimal)

Minimal single-purpose web app to generate professional PDF invoices from form inputs. Built with HTML, vanilla JS and jsPDF.

## Features
- Enter client, items, taxes
- Preview invoice
- Export to PDF (jsPDF)

## How to run
Open `index.html` in a browser.

## Notes
Includes `.kiro` directory as required by the submission.
